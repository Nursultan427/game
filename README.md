🛠️ Установка
1. Клонировать репозиторий
git clone https://github.com/Nursultan427/rl-arena.git
cd rl-arena

2. Установить зависимости
pip install pygame numpy matplotlib

▶️ Запуск
Если хочешь увидеть именно сайт то через папку с игрой открываешь index.html

Награды
| Событие               | Награда |
| --------------------- | ------- |
| Обычный ход           | `-1`    |
| Сбор монеты           | `+20`   |
| Столкновение с врагом | `-50`   |

📈 Обучение
Во время запуска:
Агент проходит 1000 эпизодов обучения
Строится график наград
После закрытия графика запускается игровое окно
